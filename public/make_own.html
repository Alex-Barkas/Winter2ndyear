how to make your own.

A guide to forking this system for your own studies.

So, you want to steal my notes site. Excellent choice. This system is built to be a minimalist, high-performance command center for your academic life, stripping away the bloat of modern learning management systems.

The architecture is simple: a static HTML/JS frontend powered by a single configuration file. No database, no backend server, no login. Just you and the code.

Prerequisites

  • A GitHub account to fork the source code.
  • Antigravity (or a similar AI coding assistant) to handle the heavy lifting.
  • Your own course syllabi and schedule.

Step 1: Clean House

Right now, this codebase is tailored to my Winter 2026 term. It has my courses (like MTHE 281 and MTHE 212) and my schedule. The first thing you need to do is clear out my data.

Navigate to public/student-config.js. This is the brain of the operation. In it, you will find a courses array and an assignments array. Delete my entries.

Step 2: Input Your Data

You don't need to write manual HTML to add your courses. Just update the configuration objects in public/student-config.js.

  • Courses: Add objects with properties for code, name, and links to your notes, textbook, and onq pages.
  • Assignments: Add your deadlines to the assignments array. The system will automatically populate the Global Schedule, Calendar, and Grade Dashboard.
"Antigravity, update my student-config.js to add a new course called 'PHYS 101' with a midterm on Feb 14th."

Use your AI assistant to make these data entry tasks trivial. It knows the structure of the file and can append or modify entries instantly.

Step 3: The Assets

There are a few key files you need to replace to truly make this yours:

  • Schedule: Replace public/schedule/schedule.pdf with your own weekly timetable. This file is linked directly from the "weekly schedule" navigation item.
  • Signature: You might want to change the "patrick taylor 2026" text in the top right. This is hardcoded in the HTML files (grep for it!) or you can ask Antigravity to variable-ize it for you.

Step 4: Customization

The site uses a defined orange accent color throughout the UI. If orange isn't your vibe, open public/style.css and find the --accent-color variable at the top. Change the hex code, and the entire site will update to match your aesthetic.

That's it. You now have a personalized, distraction-free academic dashboard. Now go study.
